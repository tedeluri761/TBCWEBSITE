                                                                                                                          TBC Concept Website Overview


ეს პროექტი არის თიბისი კონცეფციის ვებსაიტის შაბლონი, სადაც წარმოდგენილია სხვადასხვა საბანკო და არასაბანკო მომსახურება. ვებსაიტი შეიცავს პროდუქტების, შეთავაზებების, მომხმარებლის სტატისტიკის და სხვა სექციებს. შინაარსი ძირითადად ქართულ ენაზეა.

მახასიათებლები
ვებსაიტი შექმნილია იმისათვის, რომ იყოს გამართული და კარგად მუშაობს სხვადასხვა ზომის ეკრანებზე.
ენას ძირითადად მოიცავს ქართულს. სხვადასხვა პროდუქტებისა და სერვისების დეტალური აღწერა.
სექციები, რომლებიც ხაზს უსვამენ სპეციალურ შეთავაზებებსა და აქციებს.
ადვილად ხელმისაწვდომი საკონტაქტო ინფორმაცია მომხმარებლებისთვის.
გამოყენებული ტექნოლოგიები
HTML5: ვებ გვერდების სტრუქტურირებისთვის.
CSS3: ვებსაიტის სტილისთვის. stylesheet style.css გამოიყენება მორგებული სტილისთვის.
JavaScript: ინტერაქტიული ფუნქციებისთვის, რომელიც შედის script.js-ში.
Google Fonts: მორგებული შრიფტებისთვის ('Inter' და 'Georgia').
ფაილის სტრუქტურა
index.html: მთავარი HTML ფაილი, რომელიც შეიცავს ვებსაიტის სტრუქტურას.
style.css: CSS ფაილი ვებსაიტის სტილისთვის.
script.js: JavaScript ფაილი ინტერაქტიული ფუნქციონირებისთვის.
აქტივები/: დირექტორია, რომელიც შეიცავს სურათებს და სხვა სტატიკური აქტივებს, რომლებიც გამოიყენება ვებგვერდზე.
თიბისი კონცეფცია: კონცეფცია და შინაარსის შთაგონება.
ღია კოდის ბიბლიოთეკები: ნებისმიერი ღია წყაროს ბიბლიოთეკა ან გამოყენებული ინსტრუმენტი.
კონტაქტინებისმიერი შეკითხვისთვის გთხოვთ დაუკავშირდეთ:

ფოსტა: info@tbcconcept.ge
ტელეფონი: +995 32 2 27 27 00
ეს README გთავაზობთ თქვენი პროექტის ყოვლისმომცველ მიმოხილვას, მათ შორის ინსტრუქციებს დაყენების, პერსონალიზაციისა და წვლილის შესახებ. თქვენ შეგიძლიათ შემდგომ მოარგოთ ის, რომ შეიცავდეს უფრო კონკრეტულ დეტალებს საჭიროებისამებრ.


ძირითადი კომპონენტები:

საბაზისო შრიფტის ზომის დაყენება html ელემენტზე rem გამოთვლებისთვის.
კონტეინერების განლაგება:

.main: Flexbox-ის განლაგება უზრუნველყოფს ხედის სრულ სიმაღლეს.
.header: სტილისტიკა ვებსაიტის სათაურისთვის flexbox-ით განლაგებისთვის და სტილისტიკა ნავიგაციის ელემენტებისთვის.
ღილაკები:

.iconandsubscribe, .searchicon, .tbclogo: ლოგოების და ღილაკების სტილები, მაუსის ეფექტებისა და კურსორის მაჩვენებლების ჩათვლით.
ტექსტი და ტიპოგრაფია:

.subscribeP, .paragrapharrow, .digitalbank: შრიფტის სპეციფიკური ზომები, წონა, ფერები და ხაზის სიმაღლეები.
გამოსახულების და მედიის მართვა:

.Cairphoto, .slider, .slide img: გამოსახულების ზომების, ობიექტების მორგების და რეაგირების უნარის მართვა.
ინტერაქტიული ელემენტები:

.სლაიდერი: ჰორიზონტალურად გადასახვევი ზონა მოცურების შინაარსისთვის.
.slider:active: კურსორის შეცვლა ურთიერთქმედებისას.
საპასუხო დიზაინი:

მედია მოთხოვნები არეგულირებს განლაგებას, შრიფტის ზომებს და ელემენტების ხილვადობას ტაბლეტისა და უფრო დიდი ეკრანებისთვის.
ეს CSS კარგად არის სტრუქტურირებული და უზრუნველყოფს დეტალურ განლაგებას და სტილის სქემას პოტენციურად დინამიური და საპასუხო ვებსაიტისთვის. თუ გჭირდებათ დამატებითი ახსნა რომელიმე კონკრეტულ ნაწილზე ან როგორ გამოიყენოთ ეს სტილი, შემატყობინეთ!


JavaScript კოდი,  კარგად მაქვს სტრუქტურირებული და ფუნქციონალურია შეხებით და მაუსის მხარდაჭერით ჩამოსატანი სლაიდერის შესაქმნელად. სათაური დაყენებულია, რომ დაფიქსირდეს გვერდის ზედა ნაწილში მაღალი z-ინდექსით, რაც უზრუნველყოფს სხვა ელემენტებს თავზე დარჩენას, ასევე სხვა ელემენტების გადაფარვას.

სლაიდერის ლოგიკაში:

ცვლადები ინიციალიზებულია გადაადგილების მდგომარეობის (isDown), საწყისი პოზიციის (startX) და გადახვევის პოზიციის (scrollLeft) თვალყურის დევნებისთვის.
ღონისძიების მსმენელები ემატება მაუსის და შეხების მოვლენებს გადაადგილების ქცევის სამართავად:
მაუსის ჩამორთმევა და შეხების დაწყების მოვლენები იწყებს წევას.
მაუსის გადაადგილება და სენსორული მოვლენები ამუშავებენ გადაადგილების მოძრაობას.
მაუსის აწევა, მაუსის აწევა და შეხების მოვლენები ასრულებენ გადათრევას და გამოიძახებენ slideToNearestSlide ფუნქციას სლაიდერის სწორად გასწორების მიზნით.
updateIndicator() ითვლის და განაახლებს ინდიკატორის პოზიციას, რომელიც წარმოადგენს სლაიდერის მიმდინარე პოზიციას.
slideToNearestSlide() უზრუნველყოფს, რომ სლაიდერი მიიკრას უახლოეს სლაიდზე გადატანის დასრულების შემდეგ, რაც უზრუნველყოფს მომხმარებლის გლუვ გამოცდილებას.
ფუნქციონირების ან შესრულების გასაუმჯობესებლად, გაითვალისწინეთ შემდეგი:

მაუსის მოძრაობისა და სენსორული ღონისძიების დამმუშავებლების გაუქმება ან ჩახშობა მუშაობის გასაუმჯობესებლად, განსაკუთრებით ნელ მოწყობილობებზე.
ხელმისაწვდომობა: სლაიდერის ნავიგაცია შესაძლებელია კლავიატურის მართვის საშუალებების გამოყენებით და ის უზრუნველყოფს ადექვატურ გამოხმაურებას ეკრანის წამკითხველებისთვის.
